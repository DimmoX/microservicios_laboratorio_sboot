# Arquetipo Frontend

Este arquetipo provee la estructura base para aplicaciones frontend con Angular.

## Estructura sugerida

- `src/`
- `angular.json`
- `package.json`
- `tsconfig.json`
- `README.md`

Incluye configuración mínima para iniciar un proyecto Angular moderno.

## Uso

Puedes copiar esta carpeta y ejecutar `npm install` para comenzar el desarrollo.
