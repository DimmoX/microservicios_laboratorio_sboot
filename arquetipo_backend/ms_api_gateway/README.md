# Arquetipo ms_api_gateway

Este arquetipo replica la estructura y nomenclatura del microservicio original `ms_api_gateway`.

## Estructura recomendada
- `src/main/java/...` (controladores, servicios, modelos, DTOs, etc.)
- `src/main/resources/` (configuración, properties)
- `Dockerfile`
- `pom.xml`
- `README.md`
- Documentación adicional

## Uso
Este arquetipo sirve como plantilla base para crear microservicios tipo API Gateway en Spring Boot, siguiendo las mejores prácticas y la arquitectura del proyecto original.
