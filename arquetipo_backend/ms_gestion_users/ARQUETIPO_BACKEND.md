# ARQUETIPO BACKEND - ms_gestion_users

Este arquetipo está diseñado para microservicios de gestión de usuarios en Spring Boot. Incluye estructura base, ejemplos de controladores, servicios, modelos, DTOs y configuración recomendada.

## Estructura
- `src/main/java/com/gestion_users/`
  - `controller/`
  - `service/`
  - `model/`
  - `dto/`
  - `repository/`
  - `exceptionHandler/`
  - `config/`
- `src/main/resources/`
- `Dockerfile`
- `pom.xml`
- `README.md`

## Ejemplo de uso
Puedes copiar esta estructura y adaptarla para tu microservicio de gestión de usuarios siguiendo las mejores prácticas del proyecto original.
