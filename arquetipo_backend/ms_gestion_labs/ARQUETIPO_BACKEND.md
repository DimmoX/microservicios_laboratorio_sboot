# ARQUETIPO BACKEND - ms_gestion_labs

Este arquetipo está diseñado para microservicios de gestión de laboratorios en Spring Boot. Incluye estructura base, ejemplos de controladores, servicios, modelos, DTOs, repositorios y configuración recomendada.

## Estructura
- `src/main/java/com/gestion_labs/`
	- `controller/`
	- `service/`
	- `model/`
	- `dto/`
	- `repository/`
	- `exceptionHandler/`
	- `config/`
- `src/main/resources/`
- `Dockerfile`
- `pom.xml`
- `README.md`

## Ejemplo de uso
Puedes copiar esta estructura y adaptarla para tu microservicio de gestión de laboratorios siguiendo las mejores prácticas del proyecto original.
